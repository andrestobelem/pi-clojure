# Ideas

Esta carpeta guarda ideas de producto, visión y diseño que todavía no son
decisiones finales.

Usamos este espacio para pensar en voz alta y conservar contexto sin mezclarlo
con documentación operativa o investigación externa.

## Criterios

- Una idea puede estar incompleta o en exploración.
- Si una idea se valida como decisión, moverla o resumirla en el lugar
  correspondiente.
- Si una idea requiere investigación externa, documentarla también en
  `docs/research/`.
- Evitar incluir secretos, datos privados o credenciales.
